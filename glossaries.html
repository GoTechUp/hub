All Glossary Terms

URC

Universal Robotic Challenge - an annual robotic competition that GoTechUp participates in as a game host.

i2loc

The Excel sheet containing terms for a specific game level tutorial in all supported languages.

GTU

GoTechUp - company branding and the name of the game extension.

GTU Tracker

Issue-centric tracker used to identify and solve game-related problems.

Term 5

Placeholder definition for Term 5.

NPC

Non-Player Character - any character in a game that is not controlled by a player.

AR

Augmented Reality - the technology overlaying digital content in a real-world environment.

API

Application Programming Interface - allows applications to communicate and share data.

UI/UX

User Interface/User Experience - focuses on design and interaction quality in applications.

SDK

Software Development Kit - tools for building applications for specific platforms.

FPS

Frames Per Second - measures the frame rate in games and videos.

DLC

Downloadable Content - additional content for a game, available as a purchase or free download.

Gantt Chart

A visual tool used for project scheduling and tracking progress.

CRM

Customer Relationship Management - tools and strategies for managing customer interactions.

VR

Virtual Reality - immersive technology creating a simulated environment.

Cloud Sync

A method to keep data consistent across devices using cloud storage.

QA

Quality Assurance - ensures a product or service meets specific standards.

Responsive Design

Web design approach that adjusts layout based on screen size and resolution.

Gamification

The use of game-like elements in non-game contexts to engage users.

Iterative Development

A cyclical approach to software development focused on refinement.

Data Mining

The process of discovering patterns in large datasets.

CI/CD

Continuous Integration/Continuous Deployment - practices for automation in software development.

AI

Artificial Intelligence - simulating human intelligence in machines.
